## Features

- Create and manage smart tasks
- Detect phone numbers, emails, URLs, and addresses using ML Kit Entity Extraction
- Open detected actions directly from task text
- Mark tasks as completed or incomplete
- Display completed tasks with strikethrough styling
- Persist tasks locally with Room Database
- Reactive UI updates with Kotlin Flow and StateFlow
- Modern UI built with Jetpack Compose and Material 3
- Simplified Clean Architecture structure
- Clear separation between presentation, domain, and data layers
